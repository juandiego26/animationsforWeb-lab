Tranformaciones

Las transformaciones CSS3 le permiten trasladar, rotar, escalar y sesgar elementos. Una transformación es un efecto que permite un cambio de elemento de forma, tamaño y posición. CSS3 admite transformaciones 2D y 3D.

Se deben usar las siguientes unidades:

  • deg, que representa un ángulo en grados. Un círculo completo equivale a 360deg. Ejemplos: 0deg, 90deg, 360deg.
  • grad, que representa un ángulo en grado centesimal. Un circulo completo equivale a 400grad. Ejemplos: 0grad, 100grad, 400grad.
  • rad, que representa un ángulo en radianes. Un círculo completo equivale a 2π radianes, que se aproxima a 6.2832rad. 1rad es 180/π grados. Ejemplos: 0rad, 1.0708rad, 6.2832rad.
  • turn, que representa el número de vueltas del ángulo. Un círculo completo equivale a 1turn. Ejemplos: 0turn, 0.25turn, 1turn.

Transformaciones de Rotación

rotate() Define una transformación para mover un elemento alrededor de un punto fijo sin deformarlo. La cantidad de movimiento se define mediante un ángulo. Si es positivo, el movimiento serán en el sentido del reloj y, si es negativo, en sentido contrario al movimiento del reloj. Una rotación de 180° se llama punto de reflexión.


Hola Mundo! 

              HTML
              <center>
                <div class="cuadro-cont">
                  <div class="cuadro">
                    Hola Mundo!                
                  </div>       
                </div>
              </center>

              CSS
              .cuadro-cont {
                border: solid 1px #ff5733;
                width: 240px;
              }

              .cuadro-cont:hover .cuadro {
                transform: rotate(45deg);
                transition:.4s;
              }

              .cuadro {
                background-color: #7dc900;
                color: #f1f1f1;
                display: flex;
                align-items: center;
                justify-content: center;
                margin: 20px;
                height: 200px;
                width: 200px;
                transition:.4s;
              }

Cambio en la dirección del giro


Yo giro en sentido contrario!

transform: rotate(-45deg);

Rotación en los ejes X, Y y Z


transform: rotateX(180deg)
transform: rotateY(180deg)

transform: rotateZ(8turn); transition: 10s;
transform: rotateX(45deg) rotateZ(45deg)
transform: rotateY(45deg) rotateZ(45deg)

Rotación en 3D


transform: rotate3d(x, y, z, rotate)
transform: rotate3d(1, 1, 1, -45deg)

transform: rotate3d(1, 0, 1, 60deg)

Transformaciones de Traslación

translate() Define una transformación para recoloca un elemento en el eje horizontal y/o vertical. Su resultado es de tipo transform-function.

Nota: la propiedad perspective permite creara un canvas en 3D para nuestros movimientos de traslación.
Con perspective-origin podemos definir el origen de esta perspectiva. Debe ser asignado al elemento padre del que se desea mover.


Hola Mundo! 

              HTML
              <center>
                <div class="circulo-cont">
                  <div class="circulo a">
                    Hola Mundo!                
                  </div>       
                </div>
              </center>

              CSS
              .circulo-cont {
                border: solid 1px #ff5733;
                border-radius: 50%;
                display: flex;
                align-items: center;
                justify-content: center;
                height: 150px;
                width: 150px;
              }

              .circulo-cont:hover .circulo.a {
                transform: translate(100px);
                transition: 1.5s;
              }

              .circulo {
                background: #62b2e5;
                border-radius: 50%;
                color: #f1f1f1;
                display: flex;
                align-items: center;
                justify-content: center;
                height: 98%;
                width: 98%;
                max-height: 140px;
                max-width: 140px; 
                transition: .5s;
              }

Traslación en los ejes X, Y y Z


Doble Movimiento
transform: translate(100px, 100px);
Sólo Eje X
transform: translateX(100px);
Sólo Eje Y
transform: translateY(-100px);
Sólo Eje Z
transform: translateZ(80px);

Traslación en 3D


Movimiento en 3D
translate3d(30px,30px,30px);

Transformaciones de Escala

scale() Define una transformación que modifica el tamaño de un elemento en el plano 2D. Debido a que la cantidad de escalado está definida por un vector, puede cambiar el tamaño de la dimensiones horizontal y vertical a diferentes escalas.

Cuando un valor de coordenadas está fuera del rango [-1, 1], el elemento crece a lo largo de esa dimensión; cuando está dentro, se encoge. Si es negativo, el resultado es un reflejo de punto en esa dimensión. Un valor de 1 no tiene ningún efecto.

La función scale() solo se escala en 2d. Para escalar en 3D se utiliza la función scale3d() en su lugar.


Hola Mundo! 

              HTML
              <center>
                <div class="circulo-cont">
                  <div class="circulo a">
                    Hola Mundo!                
                  </div>       
                </div>
              </center>

              CSS
              .circulo-cont {
                border: solid 1px #ff5733;
                border-radius: 25%;
                display: flex;
                align-items: center;
                justify-content: center;
                height: 150px;
                width: 150px;
              }

              .circulo-cont:hover .circulo.a {
                transform: scale(1.14);
                transition: 1.5s;
              }

              .circulo {
                background: #7dc900;
                border-radius: 25%;
                color: #f1f1f1;
                display: flex;
                align-items: center;
                justify-content: center;
                height: 98%;
                width: 98%;
                max-height: 140px;
                max-width: 140px; 
                transition: .5s;
              }

Escala en los ejes X, Y y Z


Doble Escala
transform: scale(1.14,.9);
Sólo Eje X
transform: scaleX(1.5);
Sólo Eje Y
transform: scaleY(1.5);
Sólo Eje Z
transform: perspective(100px) translateZ(-25px);

Transformaciones de Sesgo

skew() Define una transformación de sesgo 2D a lo largo de los ejes X y el eje Y.

Esta transformación es un mapeo de corte (transvección) que distorsiona cada punto dentro de un elemento por un cierto ángulo en las direcciones horizontal y vertical. Las coordenadas de cada punto se modifican por un valor proporcional al ángulo especificado y la distancia al origen; por lo tanto, cuanto más lejos del origen se encuentre un punto, mayor será el valor agregado.

              <center>
                <div class="cuadro-cont">
                  <div class="cuadro">
                    Hola Mundo!                
                  </div>       
                </div>
              </center>

              CSS

                border: solid 1px #62B2E5;
                width: 240px;
              }

              .cuadro-cont:hover .cuadro {
                transform: skew(10deg);
                transition:.4s;
              }

              .cuadro {
                background-color: #FF5733;
                color: #f1f1f1;
                display: flex;
                align-items: center;
                justify-content: center;
                margin: 20px;
                height: 200px;
                width: 200px;
                transition:.4s;
              }

Punto de transformación

transform-origin - Especifica la posición de origen que se usará para la transformación. Por defecto es el lado superior izquierdo del elemento. Esta posición es usada por varias transformaciones como rotar, escalar o torcer ya que necesitan de este punto especifico como parámetro.


Hola Mundo! 

              HTML
              <center>
                <div class="cuadro-cont">
                  <div class="cuadro">
                    Hola Mundo!                
                  </div>       
                </div>
              </center>

              CSS
              .cuadro-cont {
                border: solid 1px #ff5733;
                width: 240px;
              }

              .cuadro-cont:hover .cuadro {
                transform: rotateZ(45deg);
                transform-origin: top left;
                transition:.4s;
              }

              .cuadro {
                background-color: #7dc900;
                color: #f1f1f1;
                display: flex;
                align-items: center;
                justify-content: center;
                margin: 20px;
                height: 200px;
                width: 200px;
                transition:.4s;
              }

Cambio en la dirección del giro


Yo giro en sentido contrario!
